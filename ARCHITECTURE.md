# Architecture

## Inputs
- Webcam / Image / Video / Screen

## Processing
- Computer vision (color-based fire detection / CV pipeline)
- Event trigger → alert + map update

## Outputs
- Camera overlay (bounding box + label)
- Evacuation map (fire, user, path, exits)
- Voice alert (TTS)
- AI assistant (Gemini)

## Platform
- Built on Base44 (low-code)
